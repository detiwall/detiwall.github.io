Dashboard

Painéis que mostram métricas e indicadores bastante importantes para o alcance de objetivos de uma forma visual que facilita a compreensão das informações geradas. Com as dashboards e possível fazer um monitoramento de resultados e chegar a resultados que nao se conseguiria de outra maneira.

Data Visualization

A visualização de dados como gráficos, mapas e outros elementos visuais pode ajudar imenso para descobrir alguns padrões ou anomalias no funcionamento de algo, ou na descoberta de algo totalmente novo.

O nosso objetivo

Haver uma parede de visualização para a comunidade académica, isto é, um sitio onde será possível a visualização de dados em grande escala para mais fácil estudo dos mesmos.
